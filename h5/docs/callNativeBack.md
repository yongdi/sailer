#callNativeBack
-----
###解释

调用native的返回功能的方法

###调用方法
```
Sailer.callNativeBack('', successcall, failcall);
```

###请求参数

| 参数 | 值 | 必填 |
| -- | -- | -- |
| call ID| string | 是 |
| action | callNativeBack | 是 |
| param |  | 否 |

e.g.

```
native-call:callId867912$action:$callNativeBack$

```

###返回

同总则里面的返回格式
