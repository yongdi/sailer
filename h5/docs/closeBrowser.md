#closeBrowser
-----
###解释

关闭当前活动的webView

###调用方法
```
Sailer.closeBrowser('', successcall, failcall);
```

###请求参数

| 参数 | 值 | 必填 |
| -- | -- | -- |
| call ID| string | 是 |
| action | closeBrowser | 是 |
| param |  | 否 |

e.g.

```
native-call:callId867912$action:$closeBrowser$

```

###返回

同总则里面的返回格式
